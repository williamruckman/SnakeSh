Tool: Subnet Calculator

The Subnet Calculator has separate IPv4 and IPv6 tabs. Each tab combines single-network analysis with a subnet-planning view.

Protocol Tabs

  • IPv4: tuned for IPv4 address analysis and subnet splitting.
  • IPv6: tuned for IPv6 analysis and subnet planning.

Address Analysis Section

  • Address input: CIDR input such as 198.51.100.55/24 or 2001:db8::1/64.
  • Analyze: parses the CIDR and fills the summary fields.
  • Summary fields: include network, prefix, netmask, wildcard, first and last host range, total addresses, usable address count, and host-bit count. IPv4 labels use Broadcast and Host Range; IPv6 labels use Last Address and Address Range.

Subnet Planner Section

  • Network: base network to split.
  • Target Prefix: desired subnet prefix.
  • Max: maximum number of subnets to generate in the table.
  • Generate: calculates and displays the subnet plan.
  • Results table: shows each generated subnet plus usable count, range, last/broadcast address, and netmask or prefix.

Status and Errors

The status line reports successful analysis or generation. Invalid input triggers a warning dialog and updates the status line with the same error.

Close Button

The standard dialog close button closes the tool window.
