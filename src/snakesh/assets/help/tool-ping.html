Tool: Ping

The Ping tool is a GUI wrapper around the platform ping utility. It builds the command line for the current operating system, launches it, and streams the output into the dialog.

Fields

  • Target: hostname or IP address to probe.
  • Count: number of probes to send.
  • Timeout per probe: per-packet timeout in milliseconds.
  • Packet size: ICMP payload size.
  • Use IPv6: switches the request to IPv6 mode.

Buttons

  • Run: builds and starts the ping command.
  • Stop: terminates the running process.
  • Clear: clears the output pane only.

The MTU / MSS Calculator can prefill the Ping tool's Packet size and Use IPv6 fields through its Send to Ping button. Prefilling does not start a probe and does not replace the current target field.

Output Area

The output pane shows the quoted command line first, followed by live stdout and stderr from the underlying ping process. When the process ends, SnakeSh appends the exit code.

Status and Dependencies

The status line reports whether the tool is ready, running, stopping, or finished. If the required system ping executable is missing, the tool reports that directly in the dialog.
