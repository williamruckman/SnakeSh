Context Menus

This page documents the custom SnakeSh menus that appear when you right-click or open a context-style action menu. Some entries only appear when a tab type, selection state, or protocol supports them.

Terminal View Right-Click Menu

  • Copy and Paste: copies the current terminal selection to the clipboard and immediately pastes it back into the session.
  • Copy: copies the current terminal selection to the clipboard.
  • Paste: sends the current clipboard text into the terminal session.
  • Open SFTP Tab: appears on SSH terminal tabs and requests a paired SFTP workspace that reuses the same saved session definition.
  • Disconnect: ends the active shell session but keeps the tab and visible history open so you can review output or reconnect later.
  • Log Session to File...: starts per-tab terminal logging when logging is currently off.
  • Stop Logging: stops per-tab terminal logging when it is currently on.

Command Bar Input Right-Click Menu

  • Undo: undo the last text change.
  • Redo: redo the previously undone text change.
  • Cut: cut the selected text.
  • Copy: copy the selected text.
  • Paste: paste clipboard text into the input field.
  • Delete: delete the selected text.
  • Select All: select the full command-bar contents.

SFTP Local Pane Right-Click Menu

  • Upload Selected: uploads the selected local files or directories to the remote side, warning before overwriting existing remote files when enabled.
  • New Folder...: creates a new directory inside the local folder currently open in the pane.
  • Rename...: renames the selected local file or directory.
  • Delete Selected: deletes the selected local files or directories after confirmation when delete warnings are enabled.
  • Refresh: reloads the current local directory view.

SFTP Remote Pane Right-Click Menu

  • Download Selected: downloads the selected remote items to the local side, warning before overwriting existing local files when enabled.
  • Upload Selected Local: uploads whatever is currently selected on the local side without switching focus, warning before overwriting existing remote files when enabled.
  • New Folder...: creates a new directory inside the remote folder currently open in the pane.
  • Rename...: renames the selected remote file or directory.
  • Delete Selected: deletes the selected remote items after confirmation when delete warnings are enabled.
  • Refresh: reloads the current remote directory listing.

Workspace Tab-Strip Right-Click Menu

  • Group Tabs...: assign the selected tab to a terminal group.
  • Manage Groups...: open the group manager. It is enabled only when groups already exist.
  • Ungroup Tabs: remove the current tab from its group. It appears enabled only when the tab is grouped.
  • Split Vertical (Side by Side): split the current tab into a side-by-side workspace layout.
  • Split Horizontal (Top/Bottom): split the current tab into a top/bottom workspace layout.
  • Rename Tab...: change the visible label for the life of the current session tab.
  • Detach Tab: move the current detachable terminal-style tab into its own window.
  • Reattach Tab: return a detached terminal-style tab to the main workspace.
  • Disconnect: end the active terminal session while keeping the tab, title, group membership, and on-screen history in place.
  • Close Tab: close the current tab. Active terminal and remote-viewer tabs ask for confirmation unless the warning is disabled in Settings.
  • Reconnect Session: reconnect a closed SSH, Telnet, Serial, or Local Shell tab in place so the preserved history remains visible.
  • Open SFTP Tab: appears on saved SSH tabs, even after the terminal has closed, so file transfer can still reuse the session definition.
  • Log Session to File... / Stop Logging: start or stop per-tab terminal logging.
  • Detach Viewer: detach a Linux RDP viewer from an embedded tab when supported.
  • Attach Viewer to Tab: reattach a detached Linux RDP viewer to the workspace when embedded mode is supported.

Session Tree Right-Click Menu

  • Add Session...: create a new session. The suggested folder depends on the current selection.
  • Edit Session...: edit the currently selected session when exactly one session is selected and no folders are selected.
  • New Folder: create a top-level folder.
  • New Subfolder: create a subfolder beneath the currently selected folder.
  • Rename Folder...: rename the selected folder when exactly one folder is selected.
  • Rename Session...: rename the selected session when exactly one session is selected.
  • Duplicate Session: duplicate the selected session when exactly one session is selected.
  • Bulk Edit Selected Sessions...: edit multiple selected sessions together when at least two sessions of the same protocol are selected.
  • Connect / Connect Selected Sessions: open the selected sessions.
  • Open SFTP Tab: open an SFTP workspace for a single selected SSH or SFTP session.
  • Install Public Key: launch the SSH/SFTP public-key installation helper for a single selected SSH or SFTP session.
  • Delete: delete the selected session(s) or folder(s).

Network Inspector Right-Click Menu

  • Copy Selected: copy the selected rows from the current Network Inspector tab as tab-delimited text with headers.
  • Copy All: copy every row from the current Network Inspector tab as tab-delimited text with headers.
  • Ctrl+C: keyboard shortcut for copy selected on the focused Network Inspector tab.
  • Ctrl+A: keyboard shortcut for select all on the focused Network Inspector tab.

Fast Commands Drop-Down Menu

  • Saved Fast Command entries: run the selected saved command.
  • No Fast Commands Saved: disabled placeholder shown when there are no saved entries.
  • Add Fast Command...: create a new saved command.
  • Manage Fast Commands...: open the Fast Command manager dialog.

Profiles Drop-Down Menu

  • Saved profile entries: apply the chosen workspace profile.
  • No Profiles Saved: disabled placeholder shown when there are no saved profiles.
  • Add Current Profile...: save the current workspace state as a profile, including currently open supported tool windows for optional autoload.
  • Profile Manager...: open the profile manager dialog to load, replace, delete, set defaults, and edit which tools start when a profile loads.

Scope Note

This page covers the custom menus provided by SnakeSh itself. Native operating-system file-picker menus, standard Qt text-field menus that SnakeSh does not customize, and external viewer menus from RDP, VNC, or NoMachine clients are outside the scope of the built-in guide.
