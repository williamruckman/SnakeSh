Tool: OUI Lookup

The OUI Lookup tool resolves MAC addresses or OUI prefixes against SnakeSh's bundled offline vendor snapshot.

Input

  • MAC address or OUI prefix: accepts full MAC addresses and shorter prefixes such as 00:11:22:33:44:55 or 70-B3-D5-F2-F/36.
  • Lookup: runs the lookup.
  • Pressing Enter in the input field also starts the lookup.

Result Fields

  • Vendor: resolved vendor name, Unknown when no match is found, or Unavailable when no bundled snapshot is installed.
  • Matched Prefix: the matched normalized prefix and its bit length.
  • Normalized Input: the cleaned hexadecimal form SnakeSh used for matching.

Status Behavior

The status line reports Ready, invalid-input validation messages, no-match messages, snapshot-unavailable messages, or Lookup complete. Invalid input resets the result fields and reports the validation error instead of doing a lookup.
