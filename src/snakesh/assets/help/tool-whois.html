Tool: Whois

The Whois tool performs WHOIS lookups through SnakeSh's internal lookup service and formats the result in a scrollable text view.

Fields

  • Query: domain name, IP address, or ASN.
  • Server: optional WHOIS server override. When blank, SnakeSh starts with the default referral path.
  • Timeout: lookup timeout in milliseconds.
  • Follow referrals: controls whether downstream WHOIS referrals are followed automatically.

Buttons and Output

  • Lookup: starts the WHOIS request.
  • Clear: clears the output pane.
  • Output pane: shows the query banner followed by the formatted WHOIS response text.

Status Behavior

The status line reports Ready, Running, Complete, or Failed. The dialog blocks closing while a background WHOIS lookup is still in progress.
