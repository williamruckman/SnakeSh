Tool: Dig

The Dig tool performs DNS lookups through SnakeSh's internal DNS service layer rather than launching an external terminal command.

Fields

  • Query: hostname or domain name to resolve.
  • Record type: DNS record type such as A, AAAA, MX, TXT, and other supported types.
  • Nameserver: optional resolver IP address. When blank, SnakeSh uses the system resolver path.
  • Timeout: lookup timeout in milliseconds.
  • Use TCP: switches the DNS transport from UDP to TCP.

Buttons

  • Lookup: starts the DNS lookup.
  • Clear: clears the output pane.

Output

The output pane first records the query summary line and then appends the formatted DNS result. Failures are reported inline with an error marker.

Status Behavior

The status line moves between Ready, Running, Complete, and Failed. The dialog cannot be closed while a threaded lookup is still running.
