Tool: ASN Lookup

The ASN Lookup tool performs WHOIS-based autonomous system lookups and summarizes the most common registry fields without hiding the raw response text.

Input

  • Enter an ASN as AS15169 or as a plain number such as 15169.
  • Optionally choose a specific WHOIS server if you do not want SnakeSh to start from the default referral chain.
  • Enable or disable referral following depending on how much of the registry chain you want SnakeSh to walk automatically.

Output

  • ASN shows the normalized result with the AS prefix.
  • AS Name, Organization, Description, Country, and Registry Server are extracted on a best-effort basis from the WHOIS payload.
  • Remarks displays any registry comment or remarks lines that were found.
  • The lower output pane keeps the raw WHOIS sections so you can inspect the full registry text.

Notes

WHOIS formats vary between registries. If a field is blank, that usually means the registry response did not expose that field in a consistently parseable way. The raw response remains available even when structured parsing is partial.
