Tool: Password Generator

The Password Generator creates cryptographically secure passwords using the local operating system's secure random source through Python's secrets module.

Fields

  • Length: password length for each generated password.
  • Count: number of passwords to generate in one pass.
  • Complexity: preset strength rule. Choices are Balanced, Strong, Very Strong, and Maximum.
  • Character Groups: toggles for Lowercase, Uppercase, Digits, and Symbols.
  • Include These Characters: characters that must appear in every generated password.
  • Exclude These Characters: characters that must never appear in any generated password.

Buttons

  • Generate: generates the requested passwords.
  • Copy All: copies the generated password list to the clipboard. It becomes enabled only after successful generation.

Output and Validation

  • Output pane: one generated password per line.
  • Status line: reports generation count or copy confirmation.
  • Validation errors appear as warning dialogs. Examples include selecting no character groups, conflicting include/exclude characters, or choosing a complexity level that conflicts with the enabled groups or selected length.
  • The last-used generator options are saved in SnakeSh settings and restored the next time you open the tool.
