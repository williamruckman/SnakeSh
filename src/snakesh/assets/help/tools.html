Tools

Tools launch as detached standalone processes so you can keep them visible while working in the main application without sharing the main window's UI event loop. They are meant to be operational sidecars, not modal interruptions, and they can stay open after the main SnakeSh window closes.

You can launch the same tools from the command line with snakesh tool list, snakesh tool <tool_key>, or for Ping specifically snakesh tool ping --packet-size <int> [--ipv6]. The Command Line and Launchers page in this guide lists every current top-level CLI option, every registered tool key, and the internal helper launch modes that also appear in snakesh --help. Settings, themes, and the optional tool-launch master-password gate still apply in standalone mode.

Resource Monitor

The Resource Monitor is SnakeSh's live local task-manager-style dashboard. It keeps an in-memory history from the moment you open it, then discards that history when you close the window.

  • Overview shows summary cards plus live charts for CPU, memory, disk I/O, network bandwidth, processes, dual-stack interface activity, and GPU telemetry when available.
  • Filesystems shows accessible local partitions and highlights the filesystem containing your home directory.
  • Interfaces shows active interface IPv4 and IPv6 addresses, link state, speed, and current receive or send rates, with optional offline-adapter visibility.
  • Processes provides a sortable searchable process table with End Task and Force Kill actions.
  • Settings controls offline-adapter visibility, zoom, telemetry refresh, process refresh, and chart-history length.

If a process stop action requires elevated privileges, SnakeSh can prompt for OS-level authorization before retrying the action.

Network Inspector

The Network Inspector collects local machine details and presents them on dedicated tabs for IP/interface data, routing, ARP neighbors, listening ports, and DNS configuration.

  • IP shows interface status, MAC address, MTU, speed, and bound IP addresses.
  • Routing shows route destinations, gateways, interfaces, metrics, and flags.
  • ARP shows neighbor entries and resolves MAC vendors through the bundled offline OUI snapshot.
  • Ports shows listening TCP and UDP sockets along with PID and process names when available.
  • DNS shows hostname, FQDN, nameservers, search domains, and resolver notes.

The current tab supports Copy Selected, Copy All, right-click copy actions, and keyboard copy shortcuts. Copied data is placed on the clipboard as tab-separated rows including column headers.

IP Scan

IP Scan is SnakeSh's built-in TCP connect scanner for quick host and service discovery without requiring an external scanner package.

  • Scan a hostname, single IP address, or CIDR target.
  • Choose between Common TCP 20, Common TCP 100, or a custom port list.
  • Watch progress while the scan runs and stop it if needed.
  • Review structured Hosts and Open Ports tabs instead of raw console output.
  • Use the live filter box on either tab to narrow the visible rows.
  • Click a host with open ports on the Hosts tab to jump into a host-focused Open Ports view.

File Hash

File Hash computes digests on a background worker so large files do not freeze the UI. Supported algorithms include MD5, SHA1, SHA256, SHA384, SHA512, and BLAKE2b.

  • Generate a digest for a selected file.
  • Verify against a pasted digest.
  • Verify against checksum files using plain, GNU coreutils, or BSD-style formats.

Diff Tool

The Diff Tool compares two text buffers or files side by side so you can inspect and merge text changes inside SnakeSh instead of switching to a separate comparison utility.

  • Open files from disk or paste text directly into either side.
  • Review line-level and inline highlights for changed regions.
  • Use the center gutter buttons to copy one change block left or right, or use the apply-all buttons for a full one-way merge.
  • Keep both panes aligned with synchronized scrolling, line numbers, and scrollbar overview maps.
  • Use the hidden-by-default search row to search either pane, toggle it with the Search button or open it with Ctrl+F, and target matches with Left/Right plus optional case-sensitive or regex matching.

Web Server

The Web Server tool launches either a local static-file server or a reverse proxy through the GUI instead of expecting you to manage a command in a separate shell.

  • Choose a bind address from detected interfaces or wildcard presets, or use Custom... for a manual IPv4/IPv6 address, then set the port and either Static Files or Reverse Proxy mode.
  • Use HTTP only, a manual certificate/key pair, a self-signed certificate, or guided certbot HTTP-01 TLS.
  • Load an optional intermediate or chain certificate alongside a manual certificate.
  • Point reverse proxy mode at an upstream URL and tune path-prefix, forwarded-header, extra request headers, timeout, TLS-verification, and WebSocket settings.
  • Enable or disable directory listing fallback for static-file mode.
  • Save, load, update, rename, and delete reusable web server profiles from the same tool window.
  • Monitor status and archived per-run logs from the tool window.

On Unix-like systems, SnakeSh can request GUI-based elevation when a privileged listener port or certbot challenge port is requested. Log cleanup and retention for archived web-server runs are controlled from the main Settings dialog.

Syslog / SNMP Monitor

The Syslog / SNMP Monitor receives syslog traffic plus SNMP traps and informs inside SnakeSh through a hidden helper process, then presents the data through a dedicated Settings tab, a separate Monitor tab for filters and received events, alert-rule editor, dashboard graphs, and archive-retention controls.

  • Choose a bind address from the same wildcard, loopback, discovered-interface, and Custom... pulldown pattern used by the Web Server tool.
  • Enable syslog UDP, syslog TCP, syslog TLS, and SNMP notifications independently, each with GUI-configured ports and TLS or SNMPv3 settings where relevant.
  • Watch the top-right runtime badge for an always-visible running, starting, stopped, or error indicator while the detailed status line continues to show listener and event counts.
  • Copy the current cell, row, selected rows, or whole visible columns, and export the filtered or selected result set as CSV or JSON.
  • Use the Monitor tab to filter by time, source, listener, protocol, transport, syslog severity or facility, hostname, app or tag, PID, message ID, SNMP version, notification OID, enterprise OID, varbind text, and alert state.
  • Save listener, alert, retention, filter, and layout combinations as reusable profiles.
  • Review dashboard charts for event rate, severity distribution, top talkers, top apps/tags, top trap OIDs, and alert trends.
  • Control hot-data retention, compressed archive retention, and archive-disk quota from the GUI so captured events do not fill local storage indefinitely.

Monitor profiles are included in SnakeSh settings exports, but the captured SQLite history and compressed archive files remain local application data. Privileged listener ports such as 514 and 162 can trigger GUI elevation on Unix-like systems.

Password Generator

Password Generator creates one or more passwords using the local OS secure random source through Python's secrets module.

  • Choose length, count, and a complexity preset.
  • Toggle lowercase, uppercase, digits, and symbols independently.
  • Require specific characters or exclude problematic ones.
  • Copy the full generated list at once, and reopen the tool with the last-used options restored.

OUI Lookup

OUI Lookup resolves MAC prefixes against SnakeSh's bundled offline OUI vendor snapshot. It accepts whole MAC addresses or shorter prefixes and prefers the longest matching prefix that exists in the local data.

ASN Lookup

ASN Lookup performs a WHOIS-based autonomous system lookup, normalizes plain numbers into AS-prefixed identifiers, and presents both structured summary fields and raw registry output.

Help

The Help window stays open independently so you can reference documentation while navigating the rest of SnakeSh, and it can also be launched directly as a standalone tool or from an installed launcher entry.

Other Utilities

  • Ping for reachability checks.
  • MTU / MSS Calculator for estimating effective MTU, max ping payload, max UDP payload, and TCP MSS from common overhead scenarios.
  • Dig for DNS query inspection.
  • Traceroute for path diagnostics.
  • Whois for domain and IP registration lookups.
  • Subnet Calculator for CIDR and mask planning.
