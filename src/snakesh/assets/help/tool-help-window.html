Tool: Help Window

The Help window is a non-modal documentation viewer that stays open while you continue using the rest of SnakeSh.

Layout

  • Contents list: page index on the left side.
  • Document browser: selected page content on the right side.
  • Intro line: explains that the help guide can stay visible while you work.

Behavior

  • Selecting an entry in the left index loads the corresponding bundled HTML page.
  • External links open through the default external browser.
  • The document viewer uses a fixed readable document style so help content remains legible under the app theme.
  • If the bundled manifest or a selected page is missing, the dialog shows an inline error message instead of failing silently.

When to Use It

Use the Help window as the in-app reference manual for settings, session fields, platform differences, command-line launch modes, context menus, and the individual tool dialogs documented throughout this guide.
