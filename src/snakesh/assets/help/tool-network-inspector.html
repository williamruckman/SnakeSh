Tool: Network Inspector

The Network Inspector is a read-only local network snapshot viewer. It refreshes local interfaces, routing, ARP neighbors, listening ports, and DNS data and presents the results on dedicated tabs.

Top Buttons

  • Refresh: starts a new snapshot collection pass. While a refresh is running, the button is disabled.
  • Auto Refresh: reruns snapshot collection on a repeating timer.
  • Every: refresh interval in seconds for auto refresh.
  • Privileged Ports / Processes: requests an elevated helper session for richer local listening-port and process visibility on the Ports tab.
  • Copy Selected: copies the selected rows from the active tab as tab-delimited text with headers.
  • Copy All: copies every row from the active tab as tab-delimited text with headers.

Tabs

  • IP: interface name, link status, MAC or primary type value, and address details. Child rows show per-address entries.
  • Routing: route family, destination, gateway, interface, metric, and flags/source information.
  • ARP: IP address, MAC address, interface, ARP or neighbor state, and resolved vendor name from the bundled OUI snapshot.
  • Ports: listening protocol, address family, local address, PID, and process name when available.
  • DNS: hostname, FQDN, nameservers, search domains, and resolver notes.

Copy and Selection Behavior

  • Rows support extended multi-selection.
  • Ctrl+C copies the current selection from the focused tab.
  • Ctrl+A selects all rows in the focused tab.
  • Right-click on a tab view opens Copy Selected and Copy All.

Status Line

The status line reports refresh progress, warnings from partial collectors, copy counts, and snapshot failures.

Operational Notes

  • The inspector is local-machine focused. It does not query a selected remote session.
  • Privileged visibility applies only to the Ports tab. IP, routing, ARP, and DNS data still use the normal local snapshot collectors.
  • Once elevated access is granted, SnakeSh keeps that helper session for the life of the tool window so auto refresh and manual refresh do not prompt again during the same session.
  • Visible detail varies by operating system and installed networking tools.
  • OUI vendor resolution is offline and uses SnakeSh's bundled snapshot.
